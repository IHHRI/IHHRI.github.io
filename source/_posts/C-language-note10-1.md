## 字符数组

- ```
  char word[] = {'H','e','I','I','O','!'};	
  //这不是C语言的字符串，因为不能用字符串的方式做计算
  ```

|   word[0]   |   H   |
| :---------: | :---: |
| **word[1]** | **e** |
| **word[2]** | **I** |
| **word[3]** | **I** |
| **word[4]** | **O** |
| **word[5]** | **!** |
